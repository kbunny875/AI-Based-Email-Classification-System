# 14. AI-Based Email Classification System (MailMind)

**Problem Statement:**
Users receive a large number of emails daily, making it difficult to identify important messages.

## 💡 The Solution

MailMind acts as an intelligent, automated assistant that reads, evaluates, and categorizes incoming emails without manual intervention. It solves the problem of information overload through:

1. **AI-Powered Classification:** Automatically sorts incoming messages into smart categories like *Urgent*, *Action Required*, *Informational*, or *Spam*.
2. **Urgency Scoring:** Assigns a 1–10 priority score to each email, ensuring that the most critical messages always float to the top of the user's inbox view.
3. **Key Phrase Extraction:** Summarizes emails instantly by pulling out the main topics, action items, or deadlines so the user doesn't have to read the entire thread.
4. **Smart Auto-Replies:** Suggests context-aware, professional replies to save time on routine responses.

## 🚀 Project Architecture

This repository contains the front-end application built to interface with the AI classification engine.

- **Frontend Framework:** React + Vite
- **Styling:** Custom CSS with a modern, glassmorphic UI prioritizing readability and visual excellence.

## 🛠️ Getting Started (Frontend)

To run the frontend locally, follow these steps:

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies (if you haven't already):
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to the local URL provided by Vite (usually `http://localhost:5173`).
